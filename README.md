# IIT Mandi Complaints & Feedback System

## Problem
Students in hostels and mess face multiple issues such as food quality, cleanliness, and maintenance problems.  
Currently, there is no single transparent platform where students can easily submit complaints.

## Solution
This project provides a simple web-based complaint and feedback system where students can submit issues related to hostel and mess services in a structured and digital manner.

## MVP (Minimum Viable Product)
The MVP is a custom-built complaint submission form created using HTML, CSS, and JavaScript.
